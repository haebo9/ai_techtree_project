# Reference Library

> 프로젝트 개발의 핵심이 되는 주요 공식 문서와 튜토리얼 링크입니다.

## 1. Frontend (Next.js & UI)
* **[Next.js Documentation](https://nextjs.org/docs)**: Framework 핵심 문서 (App Router, Rendering).
* **[React Flow](https://reactflow.dev/docs/introduction/)**: 스킬 트리 시각화 및 인터랙티브 그래프 구현 가이드.
* **[Shadcn/ui](https://ui.shadcn.com/)**: 프로젝트에 사용되는 UI 컴포넌트 라이브러리.
* **[Tailwind CSS](https://tailwindcss.com/docs)**: 스타일링 유틸리티 클래스 검색.

## 2. Backend (FastAPI & DB)
* **[FastAPI](https://fastapi.tiangolo.com/)**: Python 비동기 API 서버 개발 문서 및 튜토리얼.
* **[MongoDB Atlas](https://www.mongodb.com/docs/atlas/)**: 클라우드 데이터베이스 설정 및 관리 가이드.

## 3. AI Engine (Agent & LLM)
* **[Langchain](https://docs.langchain.com/oss/python/langchain/multi-agent/index)**: multi-agent 시스템 구축 가이드.
* **[LangGraph](https://langchain-ai.github.io/langgraph/)**: Stateful 에이전트 워크플로우 및 순환형 그래프 설계.
* **[OpenAI API](https://platform.openai.com/docs/introduction)**: LLM 모델 활용 및 프롬프트 엔지니어링 가이드.

## 4. Infrastructure (DevOps)
* **[Vercel](https://vercel.com/docs)**: 프론트엔드 배포 및 Edge Network 관련 문서.
* **[Docker](https://docs.docker.com/)**: 컨테이너 표준화 및 배포 환경 구축.

## 5. 기술 블로그 참고자료
