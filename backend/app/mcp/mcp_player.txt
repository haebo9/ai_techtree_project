AI TechTree MCP 설명서 
============================================================

이 문서는 Kakao PlayMCP 등록을 위한 AI TechTree 서버 설명서입니다.
(부제: "AI 개발자가 되는 길")

⭐ 1. MCP 목적 (Purpose)
------------------------------------------------------------
"막막한 AI 공부, 길을 찾아드립니다."

AI TechTree는 AI 커리어 네비게이터입니다.
초심자부터 현업 개발자까지, 자신에게 맞는 학습 경로를 찾도록 돕습니다.

❤️ 주요 가치:
- 개인화: 사용자 경력과 관심 분야에 맞춰 최적의 직무 트랙을 제안합니다.
- 구체성: 단계별 커리큘럼과 상세 학습 개념(Lv1~Lv3)을 명확히 제시합니다.
- 현장감: 최신 기술 트렌드와 한국 기업의 실제 도입 사례를 실시간으로 연결합니다.
- 유연성: 자연스러운 대화만으로 사용자의 의도를 찾아내 기술 DB와 매끄럽게 연결합니다.

⭐ 2. 도구별 상세 가이드 (Tool Guide)
------------------------------------------------------------

❤️ [1] get_techtree_survey
"사용자 성향 진단"

- 목적: 사용자의 개발 연차와 관심 분야를 파악하기 위한 설문 질문을 생성합니다.
- 사용 시점: 대화 초반, 사용자가 막연하게 학습 방법을 물어볼 때 사용합니다.
- 입력 인자: 없음

❤️ [2] get_techtree_track
"AI 직무 트랙 추천"

- 목적: 사용자 정보를 분석하여 적합한 커리어 트랙을 추천하거나 전체 목록을 보여줍니다.
- 사용 시점: 설문 후 트랙을 정하거나, 트랙 종류를 물어볼 때 사용합니다.
- 입력 인자:
    - interests: 관심 키워드 리스트 (예: ['LLM', 'backend'] 또는 ['ALL'])
    - experience_level: 개발 경력 수준 ('beginner', 'intermediate', 'expert')

❤️ [3] get_techtree_path
"트랙별 학습 로드맵 조회"

- 목적: 선택한 트랙의 단계별 커리큘럼(Step-by-Step) 전체를 조회합니다.
- 사용 시점: 특정 트랙의 구체적인 학습 순서나 과목 구성을 확인할 때 사용합니다.
- 입력 인자:
    - track_name: 트랙의 정확한 명칭 (예: 'Track 1: AI Engineer')

❤️ [4] get_techtree_subject
"과목별 상세 개념 학습"

- 목적: 특정 과목에서 학습해야 할 상세 개념(Lv1~Lv3)을 제공합니다.
- 사용 시점: 로드맵의 특정 주제(예: Vector DB)에 대해 깊이 알고 싶을 때 사용합니다.
- 입력 인자:
    - subject_name: 과목의 정확한 명칭 (예: 'Vector DB')

❤️ [5] get_techtree_trend
"실시간 기술 트렌드 검색"

- 목적: 최신 뉴스, 기술 블로그(한국), 논문 등을 검색하여 제공합니다.
- 사용 시점: 최신 기술 동향이나 실제 기업 적용 사례가 궁금할 때 사용합니다.
- 입력 인자:
    - keywords: 검색 키워드 리스트 (예: ['RAG', '실무'])
    - category: 검색 카테고리
        - 'k_blog': 한국 기술 블로그 (추천)
        - 'tech_news': 글로벌 뉴스
        - 'engineering': 구현/코드 중심
        - 'research': 연구 논문

⭐ 3. 요약 (Summary - 500자 이내)
------------------------------------------------------------
"막막한 AI 공부, 길을 찾아드립니다."

AI TechTree는 AI 학습을 시작하는 분들을 위한 커리어 네비게이터입니다. 사용자의 배경과 관심사를 분석해 최적의 직무 트랙을 추천하고, '무엇을, 어떤 순서로' 공부해야 할지 단계별 로드맵과 상세 개념을 명확히 안내합니다.

주요 특징:
- 👤 개인화: 내게 딱 맞는 커리어 트랙 발견
- 🗺️ 구체성: 기초부터 심화까지, 체계적인 커리큘럼 제공
- ⚡ 현장감: 한국 기업의 실무 사례와 최신 기술 트렌드 실시간 검색
- 💬 유연성: 자연스러운 대화로 필요한 정보에 즉시 연결

단순 검색을 넘어, 성장의 방향을 제시하는 AI TechTree와 함께 AI 개발자의 길을 시작하세요.

ps. 재능 없다고 해서 죄송합니다. 재능 없는 저도 AI 개발자가 되고 싶기에 이런 제목을 지었습니다.
