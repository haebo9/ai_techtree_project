# AI TechTree MCP Server

**AI TechTree**는 AI 개발 학습의 나침반이 되어주는 지능형 MCP 서버입니다. 사용자 맞춤형 커리큘럼 추천, 체계적인 학습 로드맵 제공, 그리고 고품질 기술 트렌드 분석 기능을 제공합니다.

이 서버를 통해 LLM은 단순한 채팅을 넘어, 신뢰할 수 있는 AI 커리어 상담가이자 전문적인 기술 리서처로 동작하게 됩니다.

## 🛠️ 도구 가이드 (Tools Guide)

AI TechTree의 지식 베이스와 상호작용하기 위해 다음 도구들을 사용합니다.

### 1. `get_techtree_track`
**맞춤형 커리큘럼 및 트랙 추천**
- **기능**: 사용자의 관심사와 경험 수준(초급/중급/고급)을 분석하여 가장 적합한 AI 기술 트랙(예: AI 엔지니어, 데이터 사이언티스트, MLOps 등)을 추천합니다.
- **사용 예시**:
  - "AI 엔지니어가 되고 싶은데 어디서부터 시작해야 할까요?"
  - "데이터 분석에 관심 있는 초보자입니다."
  - "현재 제공되는 모든 학습 트랙을 보여주세요."

### 2. `get_techtree_path`
**상세 로드맵 조회**
- **기능**: 특정 트랙의 전체 커리큘럼을 단계별(Step-by-Step)로 조회합니다. 선수 과목과 학습 순서를 체계적으로 파악할 수 있습니다.
- **사용 예시**:
  - "LLM 애플리케이션 트랙의 로드맵을 보여줘."
  - "AI 엔지니어링 과정의 전체 커리큘럼이 궁금해."
  - "파이썬 기초 다음에는 무엇을 공부해야 하나요?"

### 3. `get_techtree_detail`
**학습 주제 상세 분석**
- **기능**: 로드맵에 포함된 특정 주제(Subject)에 대해 구체적인 학습 개념(Lv1, Lv2, Lv3 범위)을 제공합니다.
- **사용 예시**:
  - "Vector DB에 대해서 구체적으로 어떤 내용을 공부해야 해?"
  - "'Advanced RAG' 모듈의 세부 커리큘럼 내용을 설명해줘."

### 4. `get_techtree_trend`
**최신 기술 트렌드 및 리서치**
- **기능**: 기술 블로그(GeekNews 등), GitHub, Arxiv, 기업 기술 블로그 등 신뢰할 수 있는 전문 도메인만을 대상으로 최신 트렌드를 검색합니다.
- **사용 예시**:
  - "Agentic AI 관련 최신 트렌드 알려줘."
  - "Reasoning Model에 대한 최신 논문 찾아줘."
  - "MLOps 구축 사례와 관련된 엔지니어링 블로그 검색해줘."

## 🚀 주요 특징 (Features)

- **개인화된 가이드**: 사용자의 실력(초/중/고급)에 맞춰 시작점과 학습 경로를 조정하여 제안합니다.
- **체계적인 학습**: 단순히 키워드를 나열하는 것이 아니라, 단계별로 구조화된 강의 계획서(Syllabus) 수준의 로드맵을 제공합니다.
- **검증된 정보원**: 검색 노이즈를 제거하고, 엔지니어와 연구자들에게 실질적으로 도움이 되는 고품질 소스만을 선별하여 트렌드를 분석합니다.

## 📦 기술 스택 (Tech Stack)
- **Protocol**: Model Context Protocol (MCP)
- **Backend**: FastAPI
- **Search**: Tavily API (Domain Filtered)
- **Knowledge Base**: Curated AI TechTree JSON
