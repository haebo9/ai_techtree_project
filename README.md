## ğŸƒ Getting Started

### Prerequisites
- Python 3.10+
- Docker & Docker Compose (for container execution)
- OpenAI API Key

### Backend Setup
```bash
cd backend
# Create Virtual Environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install Dependencies
pip install -r requirements.txt
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

# Run Server
uvicorn main:app --reload
